Rock-Paper-Scissor-Lizard-Spock

Instruction:

  • Put your input in prompt box.
  • Put 'sc' if you select scissor.
  • Put 'p' if you select paper.
  • Put 'r' if you select rock.
  • Put 'l' if you select lizard.
  • Put 'sp' if you select spock.

The rules are:

  1. Scissor cuts paper.
  2. Paper covers rock.
  3. Rock Crushes Lizard.
  4. Lizard Poisons Spock.
  5. Spock Smashes Scissor.
  6. Scissor decapitates lizard.
  7. Lizard eats paper.
  8. Paper disproves Spock.
  9. Spock vapourises rock.
  10. And as it always has been,rock crushes scissor.
